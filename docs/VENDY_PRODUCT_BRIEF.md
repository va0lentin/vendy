# VENDY — Product Brief

## 1. Vision simple

Vendy simplifie l’e-commerce comme Vinted a simplifié la vente entre particuliers, mais pour les petites boutiques, créateurs, vendeurs Instagram et vendeurs pros légers.

Vendy n’est pas simplement une marketplace.
Vendy est d’abord une infrastructure de vente ultra simple pour petites boutiques, qui peut devenir progressivement une marketplace communautaire et curatée.

## 2. Phrase centrale

Vendy permet à une petite boutique de vendre en ligne aussi simplement qu’un particulier vend sur Vinted, sans créer de site web.

## 3. Promesse principale

Publiez vos produits. Partagez votre lien. Expédiez. Vendy s’occupe du paiement, du suivi, de la validation et de la confiance.

Autre formulation :
Occupez-vous de vendre. Vendy s’occupe du reste.

## 4. Ce que Vendy résout

Aujourd’hui, beaucoup de petits vendeurs ont :
- des produits ;
- une petite audience Instagram, TikTok, WhatsApp ou locale ;
- des clients intéressés ;
- mais pas de système simple pour vendre proprement.

Ils doivent gérer eux-mêmes :
- les paiements ;
- les adresses ;
- la livraison ;
- les suivis ;
- les retours ;
- les litiges ;
- la confiance acheteur.

Vendy transforme ce parcours bricolé en un flux simple, guidé et sécurisé.

## 5. Ce que Vendy n’est pas

Vendy n’est pas :
- un Shopify bis ;
- une marketplace générique ;
- un AliExpress premium ;
- un Etsy en plus joli ;
- un Vinted d’occasion ;
- un simple lien de paiement ;
- un réseau social shopping ;
- un site builder complexe.

## 6. Différenciation

Shopify donne des outils. Vendy donne un parcours prêt à suivre.

Vinted simplifie la revente entre particuliers. Vendy simplifie l’e-commerce pour petites boutiques.

Etsy est chaleureux, mais trop large et trop bazar. Vendy doit être plus rangé, plus curaté, plus cohérent.

AliExpress est produit-prix-volume. Vendy doit être boutique-identité-confiance.

## 7. Cible vendeur

Vendy s’adresse d’abord à :
- petites boutiques locales ;
- vendeurs Instagram ;
- créateurs ;
- friperies modernes ;
- marques indépendantes ;
- micro-marques streetwear ;
- vendeurs de bijoux/accessoires ;
- vendeurs sur marchés ou pop-up stores ;
- vendeurs pros légers qui ne veulent pas créer un site.

## 8. Cible acheteur

L’acheteur arrive souvent depuis :
- un lien Instagram ;
- un lien WhatsApp ;
- une story TikTok ;
- un QR code ;
- une boutique physique ;
- une recommandation.

Il doit comprendre rapidement :
- qui vend ;
- ce qui est vendu ;
- pourquoi il peut faire confiance ;
- comment il est protégé ;
- comment suivre la commande ;
- comment signaler un problème.

## 9. Positionnement long terme

À terme, Vendy peut devenir une marketplace e-commerce complète et communautaire, où un profil peut créer plusieurs boutiques, chacune avec sa niche, son identité, ses produits, ses commandes et ses liens.

Mais au départ, le MVP doit rester simple :
une boutique, des produits, un lien, une commande, un bordereau, une validation.
