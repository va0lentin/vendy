# VENDY — Design System Guidelines

## 1. UI direction

Vendy must feel:
- premium ;
- warm ;
- modern ;
- simple ;
- reassuring ;
- mobile-first ;
- boutique-oriented ;
- curated ;
- not overloaded.

Visual inspiration :
- Airbnb for warmth, trust, white space and clarity ;
- Vinted for operational simplicity ;
- Etsy for creative boutique warmth, but without the bazaar feeling ;
- Duolingo for seller motivation, but without childish visuals.

## 2. What to avoid

Avoid :
- generic marketplace UI ;
- crowded product grids ;
- too many colors ;
- AliExpress feeling ;
- Etsy-style chaotic categories ;
- Shopify-like complexity ;
- childish gamification ;
- excessive animations ;
- casino-like rewards ;
- fake AI-generated product photos ;
- drop-shipping catalog feeling.

## 3. Color palette

Main colors :
- Vendy Coral — #F04465
- Coral Soft — #FFF0F3
- Background Warm — #FFFCFA
- Surface White — #FFFFFF
- Soft Gray — #F7F7F7
- Border Gray — #DDDDDD
- Text Primary — #1F1F1F
- Text Secondary — #717171
- Text Muted — #A3A3A3

Functional colors, used rarely :
- Trust Green — for secure payment, validation, money released, success.
- Warning Orange — for pending action, shipping deadline, reminders.
- Dispute Red — for errors, disputes, destructive actions.
- Progress Purple — for seller missions, badges, progression only.

## 4. Color rule

80% neutral :
white, warm white, gray, black.

10% Vendy Coral :
main CTAs, active links, brand moments.

5% green :
trust and validation.

3% purple :
seller motivation and progression.

2% orange/red :
warnings, errors, disputes.

The interface must never feel like a rainbow.

## 5. Typography

Use a modern sans-serif.
Recommended :
- Inter
- Geist
- SF-like system font

Hierarchy :
- Large bold title
- Short subtitle
- Clear body text
- Small muted metadata

Avoid long paragraphs inside screens.

## 7. Layout principles

- mobile-first ;
- max-width mobile preview around 390px / 430px ;
- large spacing ;
- 16px side padding minimum ;
- one main action per screen ;
- rounded cards ;
- subtle borders ;
- very soft shadows if needed ;
- high-quality image placeholders ;
- no dense dashboard tables.

## 8. Tone of voice

Short, human, reassuring.

Examples :
- Your shop is ready to sell.
- Share this link with your customers.
- Your payment is secure.
- The seller gets paid after your validation.
- Print the label or show the QR code.
- Your first order has arrived.
- One more step before your first sale.
- Vendy protects this order.

## 9. Dark mode

The prototype can include a theme setting preview, but default must be light.
Dark mode is optional and should not dominate the first prototype.

## 10. Personalization principle

Vendy is not a website builder.
Vendy is a ready-to-sell shop builder.

Allow simple personalization :
- shop name ;
- logo/avatar ;
- cover image ;
- short description ;
- accent color ;
- light/dark theme ;
- simple style preset later.

Avoid :
- complex drag-and-drop ;
- 30 fonts ;
- pixel-level editing ;
- full Shopify theme editor feeling.
