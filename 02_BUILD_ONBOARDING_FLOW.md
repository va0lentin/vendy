# 02 — BUILD ONBOARDING FLOW

Paste this after the design system is created.

---

Build the onboarding and account flow.

Important:
Vendy uses one universal account.
A user can buy and sell with the same profile.
The onboarding choice only determines the first experience.
Do not create separate permanent buyer/seller accounts.

Screens to build:
1. Welcome screen
2. Sign up / login screen
3. Intention choice screen
4. Account created screen
5. Buyer first experience redirect
6. Seller first experience redirect
7. Become a seller screen for existing buyer

Copy ideas:
Welcome headline:
"Create your shop. Share your link. Sell securely."

Subtitle:
"Vendy helps small shops turn Instagram, WhatsApp and TikTok conversations into secure orders with payment, shipping, tracking and buyer protection."

Buttons:
- "Create my shop"
- "I’m here to buy"

Trust line:
"Secure payment • Shipping label • Buyer validation"

Intention choice:
"What do you want to do first?"

Card 1:
"I want to sell"
"Create a shop, publish products and receive secure orders."

Card 2:
"I want to buy"
"Discover independent shops and track protected orders."

Small note:
"You can buy and sell with the same Vendy account."

Visual rules:
- warm, premium, minimal
- not too much text
- one main action per screen
- mobile-first
- make the flow feel real and complete on the surface
