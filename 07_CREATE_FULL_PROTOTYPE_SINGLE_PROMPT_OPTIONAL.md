# 07 — OPTIONAL SINGLE PROMPT

Use this only if you want Cursor to attempt a full first version.
The staged prompts are safer.

---

Build a high-fidelity mobile-first coded prototype for Vendy using Next.js, TypeScript, Tailwind CSS, shadcn/ui if available, Lucide icons and local mock data.

Do not build a production app.
Do not integrate real auth, Stripe, shipping APIs or databases.
Everything must be simulated but visually realistic.

Vendy is the Vinted of e-commerce for small shops:
a seller creates a shop, publishes products, shares a link, receives a secure order, gets a shipping label or QR code, ships the package, and gets paid after buyer validation.

Build all main flows:
1. Welcome / onboarding
2. Universal account sign up
3. Intention choice: buy or sell
4. Buyer discover home
5. Become seller
6. Create shop
7. Customize shop
8. Seller dashboard
9. Add product
10. Product link ready
11. Public product page
12. Public shop page
13. Secure checkout simulation
14. Order confirmed
15. Seller order received
16. Shipping label / QR code
17. Buyer tracking
18. Buyer validation
19. Dispute simulation
20. Payout status
21. Profile/settings
22. Seller missions/badges preview
23. Curated discovery after purchase

Account logic:
One universal account.
A user can buy and sell with the same profile.
Buyers can activate seller mode later.
Sellers can buy from other shops.

UI:
Premium, warm, modern, Airbnb-like clarity.
Mostly neutral.
One main brand color.
No generic marketplace.
No crowded grid.
No AliExpress/Etsy bazaar.
No childish gamification.

Colors:
- Vendy Coral: #F04465
- Coral Soft: #FFF0F3
- Background Warm: #FFFCFA
- Surface White: #FFFFFF
- Soft Gray: #F7F7F7
- Border Gray: #DDDDDD
- Text Primary: #1F1F1F
- Text Secondary: #717171
- Text Muted: #A3A3A3

Functional colors used sparingly:
green for trust, red for disputes, orange for warnings, purple for seller missions.

Use realistic mock data:
Shops:
- Lina Studio
- Maison Retro
- Atelier Pascale
- Studio Nori
- Rue Vintage

Products:
- Oversized graphic hoodie
- Handmade linen jacket
- Vintage denim coat
- Upcycled tote bag
- Minimal silver ring

Make the prototype visually complete enough to show to real sellers.
