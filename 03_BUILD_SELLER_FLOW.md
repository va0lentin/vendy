# 03 — BUILD SELLER FLOW

Paste this after onboarding is done.

---

Build the complete seller prototype flow.

Do not integrate real backend.
Use mock data.
The seller must feel guided and accompanied.

Screens to build:
1. Create Shop
2. Customize Shop
3. Shop Progress
4. Seller Dashboard
5. Add Product
6. Product Link Ready
7. Share Link
8. Seller Order Received
9. Shipping Label / QR Code
10. Payout Status
11. Seller Missions preview
12. Badge Unlocked preview

Key idea:
The seller should not feel like they are building a website.
The seller should feel like they are creating a ready-to-sell shop.

Create Shop fields:
- Shop name
- Short description
- Category
- Location optional
- Instagram handle optional

Customize Shop:
- cover image
- logo/avatar
- accent color
- light/dark theme option
- shop URL preview
- simple preview card

Shop URL examples:
- vendy.app/lina-studio
- vendy.app/atelier-pascale

Seller Dashboard:
Top:
"Good morning, Lina"
"Your shop is ready to sell."

Main card:
"Share your shop"
vendy.app/lina-studio

Buttons:
- Copy link
- Share on Instagram
- Share on WhatsApp
- Create QR code

Today card:
- 12 visits
- 1 pending order
- 0 disputes

Progress card:
- First product published
- Link shared
- Next step: receive your first order

Primary action:
"Add a product"

Add Product:
- Photos
- Product name
- Price
- Size / variation
- Stock quantity
- Description
- Shipping category: Light / Medium / Heavy

Important:
Use automatic weight logic in the UI:
- Light: T-shirt, cap, jewelry — 500g
- Medium: hoodie, jeans, dress — 1kg
- Heavy: jacket, shoes — 2kg

Product Link Ready:
Title:
"Your product is ready to sell"

Link:
vendy.app/lina-studio/oversized-hoodie

Text:
"Your customers can pay securely, track delivery and validate the order on Vendy."

Seller Order Received:
Title:
"New order received"

Show:
- buyer
- product
- price
- payment secured
- shipping deadline
- main instruction

Instruction:
"Pack the item, show the QR code or print the label, and drop off the package."

Shipping screen:
Show both options:
- QR code: no printer needed
- PDF label: print and attach

Payout screen:
Show:
- Pending buyer validation
- Available balance
- Transfer to bank button
- Money released after buyer validation

Missions:
- Add 3 products this week
- Share your shop on Instagram
- Reply to messages in under 2h
- Ship your next order quickly

Badge preview:
"Fast Shipping badge unlocked"

Visual rules:
- seller side can be lightly motivating
- gamification must be premium and subtle
- no childish game visuals
- no casino feeling
