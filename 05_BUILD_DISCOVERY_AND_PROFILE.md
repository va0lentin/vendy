# 05 — BUILD DISCOVERY, PROFILE AND SETTINGS

Paste this after buyer flow is done.

---

Build discovery, profile and settings screens.

Discovery must be curated, not chaotic.
Do not create an AliExpress/Etsy bazaar.
Do not show every possible category.
Shop identity comes before product grid.

Screens to build:
1. Buyer Discover Home
2. Shops page
3. Search page
4. Profile page
5. Settings page
6. Seller/Buyer mode switch
7. Optional seller rankings preview

Buyer Discover Home:
Title:
"Discover independent shops"

Subtitle:
"Selected shops, creators and pieces based on your taste."

Sections:
- Selected shops
- Rising shops this week
- New drops
- Top vintage shops
- Independent brands to watch
- Recently viewed

Use shop cards first, product cards second.

Search:
Tabs:
- Shops
- Products
- Categories

Filters:
- Category
- Price
- Location
- Shipping
- Verified shops
- New arrivals
- Rising shops

Profile:
Show universal account logic:
- Orders
- Followed shops
- Saved products
- Addresses
- Payment methods
- My seller space
- Become a seller / Switch to seller mode
- Settings

If seller mode active:
- My shop
- Products
- Orders
- Payouts
- Shipping
- Missions
- Badges
- Rankings
- Shop settings

Settings:
- Account
- Appearance
- Shop settings
- Payments / Payouts
- Shipping preferences
- Notifications
- Missions and badges
- Vendy rankings
- Help
- Legal
- Log out

Appearance:
- Light mode
- Dark mode
- System default

Shop personalization settings:
- cover image
- logo
- accent color
- theme
- shop bio
- shop link

Seller rankings:
Make ranking optional.
Option:
"Participate in Vendy rankings"
Description:
"Appearing in rankings can help your shop get discovered."

Ranking examples:
- Top Shops of the Week
- Rising Shops
- Most Reliable Sellers
- Top Vintage

Important:
Rankings must not be based only on revenue.
They should value reliability, reviews, shipping speed, low dispute rate and progression.
