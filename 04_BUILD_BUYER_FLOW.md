# 04 — BUILD BUYER FLOW

Paste this after seller flow is done.

---

Build the complete buyer prototype flow.

The buyer must feel safe, reassured and confident.
Do not make this flow playful.
Buyer side must be calm, premium and trust-first.

Screens to build:
1. Public Product Page
2. Public Shop Page
3. Secure Checkout Simulation
4. Delivery / Relay Point Simulation
5. Order Confirmed
6. Buyer Tracking
7. Buyer Validation
8. Report Problem / Dispute Simulation
9. Discovery After Purchase

Public Product Page:
This is what a buyer sees after clicking a Vendy link from Instagram or WhatsApp.

Show:
- product photos
- product name
- price
- shop name
- rating / trust
- short description
- size / condition / stock
- delivery estimate
- trust module

Trust module:
"Protected by Vendy"
- Secure payment
- Seller ships with tracking
- Money released after buyer validation
- Dispute support if something goes wrong

Primary CTA:
"Buy securely"

Secondary CTA:
"View shop"

Public Shop Page:
This is not a seller profile.
It is a real boutique.

Show:
- cover image
- logo/avatar
- shop name
- verified/rising badge
- short description
- location
- rating
- sales
- followers
- products
- collections
- reviews

Checkout:
Sections:
- Delivery address
- Relay point selection simulation
- Shipping method
- Payment method
- Order summary

Trust reminder:
"Your payment is held securely until you confirm reception."

Primary CTA:
"Pay securely"

Order Confirmed:
Title:
"Order confirmed"
Steps:
1. Seller prepares the package
2. Package is shipped
3. You receive it
4. You validate
5. Seller gets paid

Tracking:
Timeline:
- Payment secured
- Seller preparing
- Label generated
- Dropped off
- In transit
- Delivered
- Waiting for validation

Validation:
Title:
"Is everything OK?"
Buttons:
- "Yes, validate the order"
- "I have a problem"

Dispute:
Options:
- I did not receive the item
- The item is damaged
- The item does not match the description
- Other issue

Discovery after purchase:
Show curated shop cards, not a random product grid.

Sections:
- Similar shops
- Rising boutiques
- Shops you may like

Visual rules:
- very clear
- trust visible
- minimal
- premium
- no crowded marketplace
