# 01 — BUILD DESIGN SYSTEM

Paste this after validating Cursor's plan.

---

Now create only the design system and base layout.

Do not build the full pages yet.
Do not add real backend logic.
Do not integrate real auth, payments or shipping.

Use:
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui if available
- Lucide icons
- mock data only

Build the foundation:
1. Mobile App Shell
2. Mobile top bar
3. Bottom navigation
4. Page container
5. Primary button
6. Secondary button
7. Text input
8. Select / chip group
9. Shop card
10. Product card
11. Order card
12. Trust badge
13. Status chip
14. Progress card
15. Mission card
16. Payout card
17. Shipping timeline
18. QR code card
19. Share link card
20. Empty state
21. Basic modal/sheet

Design requirements:
- mobile-first
- premium, warm, clean
- mostly neutral colors
- Vendy Coral only for main CTAs and brand moments
- lots of spacing
- no visual clutter
- no generic marketplace feeling
- no childish gamification

Use this palette:
- Vendy Coral: #F04465
- Coral Soft: #FFF0F3
- Background Warm: #FFFCFA
- Surface White: #FFFFFF
- Soft Gray: #F7F7F7
- Border Gray: #DDDDDD
- Text Primary: #1F1F1F
- Text Secondary: #717171
- Text Muted: #A3A3A3

At the end, create a simple design system preview page showing all components with mock content.
